/*
跳转语句
break(是C语言中的关键字)
break关键字只能出现在循环结构或者switch语句中
如果break关键字出现在switch语句中,会跳出switch
如果break关键字出现在循环结构中,会跳出循环
continue(是C语言中的关键字)
continue关键字只能出现在循环结构中
如果continue关键字出现在循环结构中,
会跳过本次循环,继续下一次循环
*/
#include <stdio.h>

int main(void)
{
int i = 0;

for(i = 0; i < 5; i++)
{
    if(i == 2)
    {
        continue;
    }
    printf("i = %d\n", i);
}

return 0;

}

/*
跳转语句
break(是C语言中的关键字)
break关键字只能出现在循环结构或者switch语句中
如果break关键字出现在switch语句中,会跳出switch
如果break关键字出现在循环结构中,会跳出循环
continue(是C语言中的关键字)
continue关键字只能出现在循环结构中
如果continue关键字出现在循环结构中,
会跳过本次循环,继续下一次循环
break和continue关键字如果出现在了嵌套循环中,
会作用于最近的循环
*/
#include <stdio.h>

int main(void)
{
int i = 0, j = 0;

for(i = 0; i < 5; i++)
{
    for(j = 0; j < 5; j++)
    {
        if(i == 2)
        {
            break;
        }
        printf("(%d %d) ", i, j);
    }
    printf("\n");
}

return 0;

}
